Python requirements¶

Following are a list of core, required and optional (but useful) packages for the workshops.

Core¶

  • Python 2.6 or 2.7 (not Python 3.x [1])
  • IPython >= 0.10
  • NumPy >= 1.3
  • SciPy >= 0.7.2
  • Matplotlib >= 1.0
  • distribute (aka setuptools) >= 0.6
[1]Python 3.x is the “next generation” Python but for astronomy analysis Python 2.x is still the best choice.

Required additional packages¶

  • pip >= 1.0
  • asciitable >= 0.5.0
  • pyfits >= 2.3.0
  • pywcs >= 1.9
  • ATpy >= 0.9.4
  • APLpy >= 0.9.5 (pyparsing, pyregion, PIL, montage are optional but useful)

Useful additional packages¶

  • coords
  • vo.table
  • pyparsing
  • pyregion
  • PIL
  • Montage (compiled application and library)
