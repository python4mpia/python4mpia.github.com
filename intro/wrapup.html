Installing Python and the rest¶

In order to follow along with future sessions you need to have Python 2.6 or 2.7 installed and satisfy these Python requirements. Instructions for installation everything you need on Linux, Mac or Windows can be found in Installing a scientific Python distribution. If you have trouble installing packages feel free to contact the MPIA python mailing list.
